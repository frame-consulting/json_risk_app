# **Reports**  
In the integrated reporting tool of the JSON risk app, results of the Simulation or Batch applets can be analyzed, i.e. results can be displayed aggregated as well as exported as csv file or copied to the clipboard for further use. The applet can be opened via the main menu or directly via the Simulation applet after successful calculations , see [documentation of the Simulation applet](/docs/Simulation.md). When opening the report applet via the Simulation applet, the result data of the calculation can be evaluated directly in the report applet without saving any data. Otherwise, result data must be available on the backend in order to analyze them, that is, a batch run must have completed.

**Analyze results**


  - **Scenario independent fields**: click **`Edit fields`** -> select or deselect instrument attributes to be displayed per instrument.
  
  - **Scenario dependent fields**: click **`Edit per-scenario fields`** -> select or deselect instruments attributes to be displayed per instrument and scenario.

  - **Scenarios**: click **`Edit scenarios`** -> select or deselect scenarios to be included in the report

  - **Aggregation attributes**: click **`Edit grouping`** -> select or deselect instrument attributes in order to group and aggregate results
